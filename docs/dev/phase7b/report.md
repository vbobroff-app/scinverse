## Открытые пункты

- Часы подтверждены: будни (ЕТС) 08:50–23:50; доп. сессия выходного дня (с 01.03.2025) 09:50–19:00,
  не каждые выходные (список исключений). Точный производственный календарь — в phase7c из ISS.
- **Правка модели (после 7b):** ось перешла на **посессионную проекцию** (`sessionProjection.ts`) —
  каждая сессия равной долей, ночь схлопывается. **Выходные НЕ схлопываем** — это отдельные слоты
  (подсвечены), сессии D/W генерируются календарно на клиенте (`recentSessions`), а не из данных.
  Схлопывание выходных станет опциональным фильтром (`includeWeekends`).
- Маппинг `M/Q/Y` теперь тоже **посессионный**: левый край — сдвиг на n мес/кв/лет, но ось строится
  по торговым дням (`sessionsFrom`), разрывы схлопнуты. При >16 сессий ось прореживает подписи
  (границы недели/месяца), а `CoverageTrack` не рисует поштучные слоты/швы (>40) — чтобы не шумело.

## Итог

Панель таймфреймов управляет окном Ганта: `D1` — сегодняшняя сессия; `D{n}`/`W{n}` — последние
сессии (5 или 7 в неделю по чекбоксу «выходные»); `M/Q/Y` — календарный сдвиг; `All` — весь экстент
покрытия; календарь — произвольный диапазон со снапом к границам сессий. Ось статична, разделена по
сессиям (подписи дат + сепараторы), правый край = конец сегодняшней сессии; окно пересчитывается по
таймеру (смена суток), для `range` — фиксировано.
