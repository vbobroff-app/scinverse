## Итог

MVP тайм-лайн-фильтра и единого стандарта времени готов: интуитивный категоризированный поповер `[+]`
(дни + окно дня), расширяемая модель (сессия = атрибут площадки, режимы дополняемы правилами),
продуман будущий функционал (торговые календари, дат-точные расписания, `Set schedule`, мультибиржа —
плейсхолдеры/задел под phase 7c и Finam). Стандарт времени — системная настройка в шапке с живыми
часами. `tsc`/`vitest`/`eslint` зелёные.

Следующие шаги (в phase 7c и позже): реальные торговые календари и дат-точные расписания из ISS,
пересчёт границ сессии по расписанию конкретной биржи (CME), персист фильтра/ТЗ в `user_settings`.
